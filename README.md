# 📈 Stock Trading Simulator

A Java-based console application that simulates basic stock trading operations like buying, selling, and portfolio tracking.

---

##  Features
- View available stocks
- Buy shares
- Sell shares
- View portfolio
- Balance management system

---

## 🛠️ Tech Stack
- Java
- Arrays
- Control Statements
- OOP Concepts

---

## ⚙️ How It Works
- User selects an option (Buy/Sell/View)
- Enters stock index and quantity
- Total price is calculated using:
  price[index] * quantity
- Balance and portfolio get updated
